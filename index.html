Données

Cette carte recense les accidents corporels de la circulation en France métropolitaine. Les données sont issues de ce document : Informations sur la localisation des accidents corporels de la circulation sur 5 années - France-Métropolitaine

Légende :

  • accidents avec blessés légers,
  • accidents avec blessés hospitalisés,
  • accidents mortels.

Attention, seulement les accidents mentionnant la localisation GPS (latitude, longitude), soit 20% seulement des accidents listés dans le document, sont affichés. (J'imagine que ce sont les accidents les plus récents.)

Implémentation

Cette carte est basée sur Leaflet et quelques lignes de javascript : accidents.js

Les données issues du fichier .csv de data.gouv.fr ont été divisées en tuiles avec le script create_tiles.py

Statistique sur le rendu :aucune
